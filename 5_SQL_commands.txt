* "SHOW databases;" -> Show all databases.

* "SHOW TABLES FROM [database name];" -> Show tables from the database spacified.

* "SELECT * FROM movr.users LIMIT 10;" -> Simply query that shows the first 10 rows from the user's table inside the "movr" database.

* "CREATE DATABASE [database_name];" -> Create a database with the specified name.

* "SET database = [database_name];" -> Set to the database with the specified name as the workspace.
